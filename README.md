# BB boilerplate

It is a boilerplate of a web app. It contains 

* bootstrap 3, 
* requirejs, 
* requirejs text plugins,
* backbone.js, 
* underscore.js,
* less.js

## Run the demo

    $ git clone https://github.com/cauliturtle/bb-boilerplate.git
    $ cd bb-boilerplate/node
    $ node app.js

Open your borwser and enter `http://localhost:1337`
